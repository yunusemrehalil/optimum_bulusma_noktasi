# Optimum Buluşma Noktası — İstanbul

İstanbul'da **K** kişi için, **H** aday nokta (restoranlar, parklar) arasından en uygun
buluşma noktasını hesaplar — iki mesafe modeliyle (Öklid kuş uçuşu /
yol ağı en kısa yol) ve iki amaç fonksiyonuyla (min-sum / min-max).

- **Tüm yöntem, sonuç ve analiz:** [`report/rapor.pdf`](report/rapor.pdf)
- **Video:** <https://youtu.be/sK8VXSB-otY>

Kod düzeni: `sql/` hesaplama, `scripts/` başarım ölçümü ve grafik, `qgis/` harita,
`outputs/` çıktılar, `report/` rapor.
